Project Overview

A dual-stepper motor testing and control system developed with DRV8825 drivers and Arduino Uno. Each motor is independently controlled using push switches for start/stop operations, enabling precision calibration and prototyping.

Technical Details

  • Independent switch control for two motors
  • Microstepping mode for smooth rotation
  • Emergency stop integration
  • Designed for testing performance and motor tuning
